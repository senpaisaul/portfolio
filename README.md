# Portfolio — Abhay Singh Sengar

Live: [abhaysengar.vercel.app](https://abhaysengar.vercel.app)

Personal 3D portfolio built with React, TypeScript, Three.js, React Three Fiber, and GSAP.

## Tech Stack

- React 18 + TypeScript + Vite
- Three.js + React Three Fiber + Rapier Physics
- GSAP (ScrollSmoother, ScrollTrigger, SplitText)
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
